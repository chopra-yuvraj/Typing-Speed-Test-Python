## 🎯 Why Advanced Typing Speed Test?

As a B.Tech CSE student at VIT and B.S. Data Science student at IIT Madras, I spend countless hours typing code, reports, and assignments. I wanted a typing test that wasn't just about speed - but also about learning something new with every practice session. This application combines skill development with knowledge acquisition across diverse academic and professional topics.

### ✨ Key Features

🎯 **30+ Unique Texts** - Each carefully crafted with 200 words covering diverse topics  
📚 **Educational Content** - Learn while you type about technology, science, history, and more  
🚀 **Auto-Completion** - Tests end automatically when you complete the full text  
🎲 **Random Selection** - Different text every time you click "New Text"  
⏱️ **Flexible Modes** - Complete text or timed challenges (60s, 120s, 300s)  
📊 **Real-Time Analytics** - Live WPM, accuracy, progress, and error tracking  
🎨 **Visual Feedback** - Color-coded highlighting for correct/incorrect/current text  
💾 **Progress History** - Track your improvement over time with detailed statistics  
📁 **Data Export** - Export results to JSON or CSV for analysis  
🖥️ **Cross-Platform** - Works seamlessly on Windows, macOS, and Linux

## 🚀 Quick Start

### Prerequisites
- Python 3.6 or higher ([Download here](https://python.org/downloads/))
- No additional packages required (pure Python standard library!)
- 5 minutes of your time

### Installation

**Step 1: Clone the repository**
```
git clone https://github.com/chopra-yuvraj/Typing-Speed-Test-Python.git
cd Typing-Speed-Test-Python
```

**Step 2: Run the application**
```
python advanced_typing_speed_test.py
```

**Step 3: Start typing!**
- Choose your test mode (Complete Text or Timed)
- Click "Start Test" and begin typing
- Watch your stats update in real-time
- Test ends automatically when you complete the text!

**If you see the application window, you're ready to improve your typing! 🎉**

## 🎮 How to Use

### Test Modes

| Mode | Description | Best For |
|------|-------------|----------|
| **Complete Text** | Type the entire 200-word text | Accuracy & completion practice |
| **60 seconds** | Type as much as possible in 1 minute | Speed bursts |
| **120 seconds** | 2-minute typing challenge | Sustained performance |
| **300 seconds** | 5-minute endurance test | Long-form typing practice |

### Real-Time Statistics

- **WPM (Words Per Minute)**: Your typing speed
- **Accuracy**: Percentage of correctly typed characters
- **Progress**: How much of the text you've completed
- **Errors**: Number of mistakes made
- **Time**: Elapsed time or countdown

### Smart Features

🎯 **Auto-Completion**: In "Complete Text" mode, the test ends automatically when you finish typing the entire text  
🎲 **Random Texts**: Click "New Text" to get a different 200-word passage  
🎨 **Visual Feedback**: Green (correct), Red (incorrect), Yellow (current position)  
📊 **Progress Tracking**: See completion percentage in real-time
